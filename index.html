    About PassGuard

    PassGuard is a privacy-first, client-side password generator designed to help users create strong, unique passwords quickly.

    It offers configurable rules so users can choose length and which character types to include.

    The app provides a real-time strength meter that gives immediate feedback on password quality.

    Features

    • Length and character-type controls
    • Presets and quick actions
    • Strength meter with breakdown
    • One-click copy and .txt file export for generated passwords
    • Light and dark themes with system preference support
    • PassGuard stores generated passwords only on your local device, keeping them private and under your control
